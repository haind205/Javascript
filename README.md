# 📋 Todo App

Ứng dụng quản lý công việc đơn giản được xây dựng bằng HTML/CSS/JavaScript thuần.

## Tính năng

- ➕ Thêm công việc mới
- 🗑️ Xóa công việc
- 🔢 Đếm số lượng công việc
- ✨ Giao diện đẹp, animation mượt mà
- 📱 Responsive cho mobile

## Cách chạy

Chỉ cần mở file `index.html` trong trình duyệt.

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (Vanilla)

## Cấu trúc project

```
todo-app/
├── index.html   # Giao diện chính
├── style.css    # Stylesheet
├── app.js       # Logic ứng dụng
└── README.md    # Tài liệu
```
